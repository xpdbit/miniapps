## 常用运维命令

```bash
# 容器状态
docker ps --format 'table {{.Names}}\t{{.Status}}'

# 查看日志
docker logs -f --tail 100 dashboard-api

# 重启服务
docker restart dashboard-api

# 部署更新
cd /opt/ftg/tools/deploy && bash scripts/deploy.sh

# 快速修复（不重建镜像）
cd /opt/ftg/tools/deploy && bash scripts/quick-fix.sh

# 磁盘空间
df -h /

# 进入容器
docker exec -it dashboard-api sh
```

## SSL 证书

- 当前使用 Let's Encrypt 证书
- 自动续期：certbot renew (cron)
- 证书路径：`/opt/ftg/tools/deploy/nginx/ssl/`

## 已知问题

- Dashboard Admin API Docker 镜像使用 bcryptjs（纯 JS）替代原生 bcrypt，避免 Alpine/glibc 跨平台编译问题
- ECS 带宽仅 3MB/s，Docker 镜像构建可能较慢（约 5-10 分钟）
- MySQL 健康检查偶尔超时，重启后恢复
