# 云函数 API 参考

> 🚫 **废弃文档** — 本文档描述的是已弃用的 CloudBase 云函数架构 (2025)。
> 当前架构已完全迁移至 Express REST API（servers/ftg-server），所有云函数功能已由 REST 端点替代。
> 当前 API 路由定义请参考 `servers/ftg-server/src/routes/`（16 个路由模块）。
> 小程序客户端通过 `apps/ftg-miniapp/src/services/httpClient.ts`（JWT 自动携带）与后端通信。
