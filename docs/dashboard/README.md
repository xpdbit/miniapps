# dashboard — 统一管理后台

集中管理所有微信小程序项目的管理后台。

## 技术栈

- **前端**: React 19 + Vite + Ant Design
- **状态管理**: Zustand
- **Admin API**: Express (独立进程，端口 3001)
- **ORM**: Prisma

## 核心功能

- 项目管理 (多小程序统一管理)
- 用户管理
- 食物记录审查
- 主题 Class 系统管理
- 主题模板编辑
- 数据统计看板
- API 密钥管理

## 架构

双进程架构：
- **Vite 前端** (端口 5173): 管理界面 SPA
- **Admin API** (端口 3001): 后台数据接口

## 页面列表 (15 个)

包含用户管理、食物记录、主题管理、Class 系统、统计面板等页面。
