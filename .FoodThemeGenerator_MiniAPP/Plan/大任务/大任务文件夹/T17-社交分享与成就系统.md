# T17: 社交分享与成就系统

- **波次**: 3
- **预估时长**: 4-5 小时
- **依赖**: T7, T11
- **阻塞**: T18

## 做什么

**A. 社交分享:**
1. 实现分享卡片生成（Canvas绘制）: 750x1000px，内容含主题合成图 + 食物名称 + 游戏描述 + 小程序码 + 引导文案
2. 实现微信分享: 使用 `<button open-type="share">` 触发，自定义分享标题和图片
3. 实现朋友圈分享: 生成带小程序码的图片
4. 分享统计: 记录每条记录的分享次数

**B. 成就系统:**
1. 定义10个成就配置:
   - 🥚 初次尝试: 完成第一次食物记录
   - 📸 美食猎人: 累计记录10种不同食物
   - 🔥 七日连击: 连续7天打卡
   - 🌍 城市探索者: 在3个不同城市打卡
   - 🏆 百食斩: 累计100条记录
   - 🍖 肉食爱好者: 记录20次肉类
   - 🥗 素食达人: 记录20次蔬菜类
   - ⭐ 星露谷居民: 使用星露谷主题10次
   - 🌑 饥荒幸存者: 使用饥荒主题10次
   - 🗺️ 全域覆盖: 累计10个地点打卡
2. 实现成就检查引擎（云函数 `checkAchievement`）: 事件驱动 + 条件判断 + 进度更新
3. 成就解锁通知: 推送模板消息或应用内弹窗动画
4. 创建成就展示页面（`src/pages/achievements/`）: 网格布局 + 进度条 + 已解锁/未解锁区分

## 绝对不能做

- ❌ 不要在每次页面加载时全量检查所有成就
- ❌ 不要在分享卡片中暴露用户隐私
- ❌ 不要让成就检查阻塞主要操作

## 推荐Agent配置

- **类别**: `deep` — 两个子系统都有复杂度
- **技能**: `[]`
