# T1: 项目基础设施搭建

- **波次**: 1
- **预估时长**: 3-4 小时
- **依赖**: 无
- **阻塞**: T2, T3, T4, T5, T6, T7, T8, T9

## 做什么

1. 使用 Taro CLI 初始化 Taro 4.x + React + TypeScript 项目
2. 配置 `@tarojs/plugin-platform-weapp` 微信小程序插件
3. 配置 `project.config.json`（appid、云开发环境ID、编译设置）
4. 初始化 CloudBase 环境：在 `app.ts` 中调用 `wx.cloud.init({ env: 'your-env-id', traceUser: true })`
5. 创建项目目录结构（src/pages/, src/components/, src/utils/, src/types/, src/services/, cloudfunctions/）
6. 配置 TypeScript 严格模式（`strict: true`, `noImplicitAny: true`, `strictNullChecks: true`）
7. 配置 ESLint + Prettier（禁止 `any`、禁止 `console.log` 在生产代码）
8. 配置分包策略（主包仅首页+核心，子包按功能模块划分）
9. 设置环境变量管理（`.env.dev`, `.env.prod`，包含 CloudBase 环境ID、AI服务endpoint）
10. 配置 `app.config.ts`（路由、tabBar、窗口样式、权限声明 camera/location）
11. 创建基础全局样式（CSS变量主题色、字体、间距系统）
12. 编写 README.md（项目说明、启动步骤、目录结构说明）

## 绝对不能做

- ❌ 不要使用 `@ts-ignore` 绕过类型检查
- ❌ 不要在主包中放置任何图片资源（统一放子包或云存储）
- ❌ 不要创建不必要的工具函数（避免过度抽象）
