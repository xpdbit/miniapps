# T16: 主题画廊与管理系统

- **波次**: 3
- **预估时长**: 3-4 小时
- **依赖**: T7
- **阻塞**: T18

## 做什么

1. 创建主题画廊页面（`src/pages/gallery/`）:
   - 网格布局，每个主题一个预览卡片
   - 每个卡片显示: 预览图、名称、游戏来源、使用次数
   - 主题详情弹窗: 大图预览、描述、使用主题按钮
2. 实现主题管理系统:
   - 当前默认主题标记（星标或高亮边框）
   - 切换默认主题
   - 主题排序: 按使用次数、按添加时间
3. 主题加载优化: 缩略图懒加载、模板图片预加载
4. 实现主题预览功能: 用示例图片快速合成预览效果
5. 预留主题扩展接口: 新增主题只需添加配置JSON和模板图片

## 绝对不能做

- ❌ 不要把所有主题模板图片打包到小程序包中
- ❌ 不要在列表中加载高清大图

- **类别**: `visual-engineering` — 画廊页面需要美观的网格布局
- **技能**: `[]`

- [ ] 主题预览功能正常
- [ ] 默认主题切换功能正常

- [ ] 主题详情弹窗正常显示
