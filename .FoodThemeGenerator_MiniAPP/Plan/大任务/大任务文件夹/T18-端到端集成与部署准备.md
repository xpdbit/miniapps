# T18: 端到端集成与部署准备

- **波次**: 3
- **预估时长**: 4-5 小时
- **依赖**: T14, T15, T16, T17
- **阻塞**: F1-F4（最终验证波次）

## 做什么

1. 跨模块集成验证: 拍照 → AI识别 → 主题合成 → 记录保存 → 打卡 → 分享（全链路无断点）
2. 验证数据一致性: food_records与checkins数量匹配、用户统计字段与实际数据匹配
3. 验证权限流程: 相机权限、GPS权限被拒绝后的降级体验
4. 错误边界实现: 全局错误捕获 + 页面级ErrorBoundary + 云函数超时处理
5. 性能优化收尾: 图片懒加载确认、分包配置确认（主包<2MB）、云函数冷启动优化、setData调用优化
6. 部署配置: 云函数部署、数据库索引部署、云存储CORS配置、环境变量配置
7. 小程序审核准备: 小程序简介、类目说明、测试账号、隐私政策页面、关闭开发调试模式
8. 构建生产版本: `npm run build:weapp -- --env production`
9. 真机测试: iOS和Android各一台

## 绝对不能做

- ❌ 不要在审核版本中包含开发调试功能
- ❌ 不要在审核前不自行真机测试
- ❌ 不要忘记配置隐私政策页面
